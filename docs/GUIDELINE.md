# スチームコモンズ 物品管理システム 利用規約（草案）
施行日: 2025-01-01 / 最終改定日: 2025-01-01

### 1. 目的

- 本規約は、「スチームコモンズ」（以下「本施設」という。）における放置物品問題を円滑に解決することを目的とするものである。
- 本規約の目的は、ユーザーが快適かつ公平に作業スペースを利用できる環境を維持することである。
- 本サービスは、「スチームコモンズ 物品管理システム」の名称で提供されるものである。
- 本規約は、本サービスの提供条件およびユーザーと本施設との間の権利義務関係を定めるものである。
- ユーザーは、本サービスを利用するにあたり、本規約の全文を確認しなければならない。


ユーザーは、本規約に承諾の上、本サービスの利用を申し込むものとし、本サービスの申し込みをした場合、本規約を承諾したものとみなします。

### 2. 定義

「本サービス」とは、本施設が「スチームコモンズ 物品管理システム」の名称で提供する、物品の登録および管理に関する一切のサービスおよびそれを提供するための情報システムをいう。

「ユーザー」とは、本規約に同意の上、本サービスの利用登録をした個人をいう。

「登録物品」とは、ユーザーが本サービスを利用して、所有者情報および明け渡し日を登録した私物を指す。

「管理者」とは、本施設の運営を担当し、本サービスの管理権限を有する者を指す。

「QRコード」とは、物品登録フォームにアクセスするために、本施設が提供する個別の識別コードを指す。

「明け渡し日」とは、ユーザーが登録物品を本施設から撤去する最終期限の日を指す。

### 3. システムの利用

本施設内に私物を設置するユーザーは、本規約に同意の上、所定の方法で本サービスに物品を登録しなければならない。

物品の登録は、本施設内に掲示されたQRコードを読み取り、表示されるフォームに必要事項（氏名、連絡先メールアドレス、物品の情報、写真、明け渡し日など）を入力することで完了する。なお、登録できる物品は第7条（禁止事項）の定めに従うものとする。

ユーザーは、本条第2項に定める写真について、登録物品が施設内のどこに置かれているかが明確に判別できるよう、周囲の状況も含めて撮影するものとする。

前各項の定めにかかわらず、管理者は、登録申請された物品が施設の利用目的にそぐわない、スペースを過度に占有する、その他管理上不適切であると判断した場合、当該物品の登録を承認しない（受け付けない）ことができる。

前項に基づき管理者が登録を承認しなかった場合、ユーザーは当該物品を速やかに本施設から撤去しなければならない。これによってユーザーに生じた損害について、管理者は一切の責任を負わない。

ユーザーは、登録情報が正確であることを保証するものとする。登録情報に変更があった場合は、速やかに更新しなければならない。

### 4. 個人情報の取り扱い

管理者は、本サービスの運営にあたり取得したユーザーの個人情報（氏名、メールアドレス等）を、本サービスの管理・運営（リマインダー通知、緊急時の連絡等）の目的でのみ利用する。

法令に基づく場合を除き、ユーザーの同意なく個人情報を第三者に開示または提供することはない。

取得した個人情報は、登録物品の管理期間終了後1年間保管し、その後速やかに削除する。個人情報の漏洩、滅失、毀損等を防止するため、適切な安全管理措置を講じる。ユーザーは、個人情報の開示、訂正、利用停止、削除を希望する場合、管理者（メールアドレス）まで連絡するものとする。

### 5. 登録物品の管理

ユーザーは、自らの責任において登録物品を管理するものとする。

ユーザーは、設定した明け渡し日までに、登録物品を本施設から撤去する義務を負う。

本サービスは、明け渡し日の到来前に、登録されたメールアドレス宛にリマインダー通知を自動送信する。ユーザーは、この通知を受信できるよう、メール設定を適切に行うものとする。
なお、リマインダー通知は明け渡し日の3日前および当日に、登録されたメールアドレス宛に自動送信される。

ユーザーは、やむを得ない事情がある場合、明け渡し日の延長を申請することができる。延長申請は所定のフォームから行い、管理者が承認または否認する。管理者が延長申請を承認した場合、明け渡し日を延長する。否認の場合は、ユーザーは速やかに物品を撤去しなければならない。

物品の設置期間は原則として年度末（3月31日）までとする。年度末が近づいた場合、管理者は全ユーザーに一斉通知を行い、明け渡し日を周知する。年度末までに撤去されない物品は、管理者が任意に処分できるものとする。

撤去完了または管理者による処分が確認された物品の登録データは、管理シートからアーカイブ用シートに移動し、過去の記録として保管する。アーカイブ後は主要情報のみを保持し、個人情報は速やかに削除する。

### 6. 明け渡し日を超過した物品の取り扱い

明け渡し日を超過しても撤去されない登録物品について、管理者は、当該物品の所有権がユーザーによって放棄されたものとみなし、ユーザーへの事前の通知なく、任意に移動、保管、または処分（廃棄、再利用を含む）することができる。

前項の措置に関し、ユーザーは一切の異議を申し立てることができず、管理者はユーザーおよび第三者に対して何らの責任も負わないものとする。

### 7. 禁止事項

ユーザーは、本サービスの利用にあたり、以下の各号に定める行為を行ってはなりません。

- 虚偽の情報を登録する行為
- 他人の情報を無断で使用して登録する行為
- 本サービスの運営を妨害する行為
- 本施設内に危険物、違法物、その他管理者が不適切と判断する物品を設置する行為
- その他、法令または公序良俗に反する行為

### 8. 免責事項

本サービスの障害、メンテナンス、通信回線の異常、その他やむを得ない事情により、予告なく本サービスの提供を中断または停止する場合がある。

本サービスの利用または利用不能によってユーザーに生じた損害について、管理者に故意または重過失がある場合を除き、一切の責任を負わないものとする。

### 8-1. エラー・障害時の対応

- システム障害、通信エラー、その他予期せぬ事象が発生した場合、管理者は速やかに復旧対応を行うものである。
- 登録・通知・アーカイブ等の処理に失敗した場合、管理者はユーザーに状況を通知し、必要な対応を協議するものである。
- ユーザーは、エラー発生時に管理者の指示に従うものとする。

### 9. 規約の変更

管理者は、必要に応じて本規約を変更できるものとします。

規約を変更した場合、管理者は本施設内への掲示その他適切な方法でユーザーに通知します。通知後、ユーザーが本サービスを利用した場合は、変更後の規約に同意したものとみなします。
