## プルリクエストの仕様

ブランチでの作業が完了し、developブランチにマージしてほしい場合は、プルリクエストを作成してください。

1.  GitHubのリポジトリページにアクセスし、「Pull requests」タブから「New pull request」をクリックします。
2.  `base` ブランチが `develop`、`compare` ブランチが自分の作業ブランチになっていることを確認します。
3.  変更内容を確認し、「Create pull request」をクリックします。
4.  テンプレートに従って、プルリクエストのタイトルと詳細を記述します。
    -   **概要**: 変更の目的を簡潔に書きます。
    -   **変更点**: 具体的な変更内容をリストで書きます。
    -   **関連イシュー**: 関連するIssueがあればリンクします。
    -   **確認事項**: チェックリストを埋めて、セルフレビューを実施します。
5.  「Create pull request」をクリックして、作成を完了します。
